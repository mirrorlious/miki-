## 更新日志

### 2026-06-08｜产品结构与性能稳定性更新

本次更新主要围绕「背诵效率」「浏览性能」「知识组织」「批量制卡」和「目录管理」进行重构。项目定位从单纯的 Anki 卡片管理工具，进一步调整为面向法硕复习的卡片背诵工作台。

---

### 一、浏览页性能优化

* 优化 `/browse` 页面在大量卡片场景下的卡顿问题。
* 修复因全局 `data` 频繁更新导致浏览页反复重算目录树和卡片列表的问题。
* 优化目录树、卡片筛选、预览区域的渲染依赖，减少无关状态变化带来的重复渲染。
* 保留 HTML 预览功能，同时增加智能 / HTML / 文本预览切换，兼容不同来源的 Anki 卡片。
* 增加每行卡片数量切换功能，支持每行 1 / 2 / 3 / 4 张卡片。
* 优化全屏 / 宽屏模式下浏览页主体区域的布局表现。

---

### 二、云同步策略调整

* 修复登录状态下 Firestore 实时监听反复报错导致页面卡顿的问题。
* 将云同步从强实时监听调整为更安全的同步策略。
* 保留账号登录、退出、本地数据使用、手动同步、自动写入等核心能力。
* 当云端同步失败时，会暂停自动重试，避免页面持续卡死。
* 本地数据仍可继续使用，不影响背诵和卡片管理。

---

### 三、浏览目录管理增强

* 浏览页左侧目录增加三点菜单操作。
* 支持在目录节点上进行：

  * 添加卡片
  * 创建子目录
  * 编辑 / 分组
  * 删除目录
  * 移动目录
  * 合并目录
  * 导出目录数据
* 解决部分卡片放错区域、章节需要合并、父级目录已有卡片但仍需创建子目录等场景。
* 移动和合并目录时同步处理 deck 信息与 Anki 导入来源路径，减少刷新后目录回退的问题。

---

### 四、重复卡片清理能力

* 浏览页增加单张卡片删除能力。
* 支持用户自行判断重复卡、异常卡、HTML 原文卡后手动删除。
* 删除卡片时同步清理对应复习记录。
* 不进行自动批量去重，避免误删有效卡片。
* 清理权限交由用户控制，保证数据安全。

---

### 五、图谱页重构：专题筛卡器

* `/map` 页面从传统思维导图调整为「拖拽式专题筛卡器 / 知识拼图 / 法硕考点雷达」。
* 支持从左侧元素库选择知识元素，并组合成专题筛选条件。
* 元素库包含：

  * 章节元素
  * 考点元素
  * 遮盖词元素
  * 结构类型
  * 复习状态
* 条件支持三种逻辑：

  * AND：必须包含
  * OR：任一包含
  * NOT：排除
* 右侧实时筛选符合条件的卡片。
* 增加共同点总结、主要集中章节、命中标签、复习状态展示。
* 结果区支持列表 / 卡片两种显示方式。
* 默认限制渲染结果数量，避免大量卡片导致卡顿。

---

### 六、图谱页性能优化

* 对卡片内容预先生成 searchableKey，避免每次筛选重复解析 HTML。
* 对法硕关键词、结构类型、复习状态进行预处理。
* 减少大数据量下的字符串归一化和重复匹配。
* 元素库分组显示，避免一次性渲染过多元素。
* 优化 3000+ 卡片场景下的图谱页打开速度和筛选响应速度。

---

### 七、首页重写：今日学习台

* 首页从「卡组管理后台」调整为「今日学习台」。
* 首页核心目标变为回答：今天应该先学什么？
* 强化：

  * 今日待复习
  * 薄弱卡数量
  * 新卡池
  * 今日完成率
  * 推荐学习任务
* 工具箱收纳到抽屉中，减少首页信息干扰。
* 资料库管理默认弱化，避免首页被卡组表格占满。
* 首页重点从“我有多少卡片”转向“我现在该做什么”。

---

### 八、统计页重写：学习诊断中心

* 统计页从卡组表格改为学习诊断中心。
* 新增学习诊断维度：

  * 今日待复习
  * 今日已完成
  * 薄弱卡数量
  * 新卡数量
  * 本周完成率
  * 掌握率
* 新增图表化区域：

  * 最近 7 天复习趋势图
  * 卡片状态分布图
  * 卡组压力排行
  * 学习热力图
* 详细卡组表格移动到底部，作为辅助数据查看。
* 页面目标调整为回答：

  * 最近学得怎么样？
  * 哪些地方拖后腿？
  * 下一步应该优先复习哪里？

---

### 九、添加卡片与批量制卡优化

* 修复添加卡片页面乱码问题。
* 优化添加卡片页文案与按钮显示。
* 批量导入页调整为更明确的批量制卡入口。
* 支持模板选择，保留普通模板与 HTML 模板。
* 新增选择题 HTML 模板：

  * 正面模板
  * 反面模板
  * CSS
  * JavaScript 控制逻辑
* 支持选择题卡片的交互式显示、答案选择、统计记录和控制中心。
* 保留外部 AI 制卡器入口。

---

### 十、界面与交互细节

* 顶部导航新增图谱入口。
* 全屏 / 宽屏模式下，主要页面主体区域能跟随变宽。
* 浏览页旗色筛选合并为下拉按钮，减少工具栏拥挤。
* 浏览页卡片预览增加模式切换，提高兼容性。
* 目录树空分支减少，避免显示大量 0 卡片目录。
* 多处拆分组件补齐缺失 import，修复运行时 ReferenceError。

---

### 当前说明

本版本重点是完成产品结构整理和核心功能稳定化。
目前专题筛卡器可以筛出卡片并跳转学习页，但 Study 页面暂未完全接入“只复习当前筛选 cardIds”的专题队列。后续可继续补充专题队列读取逻辑，使“开始复习这组卡”真正进入精准复习模式。
